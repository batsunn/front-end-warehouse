
### 3.播放音乐评论

> 请求方式：get
>
> 接口：/comment/hot
>
> 参数1：type：0      类型：Int
>
> 参数2：id      这个ID是之前播放音乐的id    类型：Int

> 正确的实例：https://autumnfish.cn/comment/hot?type=0&id=298248

##### 实例：

```json
{
    "topComments": [],
    "hasMore": true,
    "hotComments": [
        {     。。。。   }
    ],
    "total": 40,
    "code": 200
}
   
```
